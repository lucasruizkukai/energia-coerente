# TGR Structure Draft

## Objetivo

Definir a estrutura operacional inicial do metodo TGR dentro do sistema.

Esta versao e um rascunho de arquitetura funcional.
Ela serve para organizar:

- protocolos
- biometros
- graficos
- fichas
- referencias

Depois os PDFs refinam cada modulo.

## Estrutura do metodo

## Regra transversal

Todos os protocolos do TGR devem incluir um bloco de leitura de chakras.

Esse bloco precisa permitir:

- listar os chakras avaliados
- marcar quais estao em harmonia
- marcar quais estao em desequilibrio
- registrar observacoes sobre cada centro quando necessario
- relacionar os desequilibrios com a leitura principal do protocolo

Campos-base recomendados para esse bloco:

- `chakras_em_harmonia`
- `chakras_em_desequilibrio`
- `leitura_chakras`
- `observacoes_chakras`

### 1. Visao geral

Pagina-resumo do TGR.

Deve mostrar:

- descricao do metodo
- protocolos disponiveis
- quantidade de biometros
- quantidade de graficos
- fichas disponiveis
- atendimentos usando TGR

### 2. Protocolos

Cada protocolo precisa ter:

- nome
- objetivo
- indicacoes de uso
- etapas
- campos de leitura
- biometros relacionados
- graficos relacionados
- ficha relacionada
- observacoes terapeuticas

## Protocolos mapeados

- expansao de consciencia

- foco do protocolo

- bloqueios observados
- pontos de reorganizacao
- resultado da analise
- observacoes

- leitura de energia vital

- desgaste

Campos-base sugeridos:

Funcao no sistema:

Campos-base sugeridos:

- resposta observada
- observacoes

Campos-base sugeridos:

- necessidade de protecao

- observacoes

- observacoes

### Psicoemocionais

Funcao no sistema:

### Chakras

- leitura dos centros energeticos

Campos-base sugeridos:

- observacoes

## Estrutura recomendada de tela por protocolo

Cada protocolo deve abrir uma pagina com estes blocos:

1. Cabecalho

- nome
- descricao curta
- finalidade

2. Campos de analise

- leituras
- escalas
- observacoes
- bloco transversal de chakras

3. Ferramentas relacionadas

- biometros
- graficos
- fichas

4. Aplicacao no atendimento

- botao ou acao para associar ao atendimento do cliente

## Estrutura recomendada de tela por atendimento TGR

- metodo

- intervencoes
- evolucao
